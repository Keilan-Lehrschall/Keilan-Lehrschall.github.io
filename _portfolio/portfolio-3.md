In this project, my friend Jaylan and I programmed the game Blackjack in Javascript for reflection use during the AP CSP exam. The game includes functions that consist of drawing cards, choosing to hit and fold based on text inputs, and incorporates a pseudo-AI that draws cards and competes with the player by using while loops and if-statements to track the scores of both players.

The project used while loops, user input, if statements, and arrays to develop the turn sequences, user options, card options, and pseudo-AI respectively.

This project taught me and Jaylan how to collaborate and work as pair programmers more effectively, as well as deepened our understanding of Javascript functions, parameters, and while loops.

We also learned how to better solve disagreements between team members and to utilize while loops to make an autonomous opponent in-game.
