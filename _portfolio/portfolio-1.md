In this project, my friend Jaylan and I made a Javascript game program similar to a typical Pokemon game. The game is turn based and allows players to use Pokemon to fight a pseudo-AI opponent with the exact same pokemon, both of which can use attacks, stat-lowering moves, and switches.\
The project uses lots of parameters and arrays in order to create the many Pokemon used in the program (pseudo-objects), as well as while loops, random number generation, an advanced calculation method that takes in array values to calculate damage, and array-mutating functions to keep track of Pokemon health and current battle status.\
This project taught me better communication and collaboration skills, research skills (since we had to search specific stats and calculations for the simulator to work as intended), and budgeting workloads between both programmers.\
Some challenges we overcame include improving code readability, disagreement resolution, and frequent loop and parameter errors.
